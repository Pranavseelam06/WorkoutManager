# Fitness Manager (Python Console App)

A menu-driven Python console application that allows users to create a fitness profile, log daily workouts, analyze workout data, and generate performance insights.

##  Features

- Interactive menu-driven interface
- User profile setup with selectable fitness goals
- Daily workout logging with strict input validation
- Support for multiple workout types and intensity levels
- Workout analysis including:
  - Total workout time
  - Average time per day
  - Time spent per workout type
  - Intensity distribution
- Derived insights such as:
  - High-intensity workout days
  - Long-duration workout days
- Ability to reset all data during runtime
- Graceful program exit
- Saves information for future usage
